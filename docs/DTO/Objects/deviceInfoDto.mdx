# Device Info DTO 

| Name           | Type       | Required           | Description                                   |
|----------------|------------|--------------------|-----------------------------------------------|
| `phone`        | `string`   | :white_check_mark: | User's phone number associated with the device. |
| `countryCode`  | `string`   | :white_check_mark: | Country code associated with the phone number. |
| `deviceID`     | `UUID`     | :x:                | Unique identifier of the device.             |
| `deviceType`   | `string`   | :x:                | Type of device, e.g., `Android` or `iOS`.    |
| `appVersion`   | `string`   | :x:                | Version of the application installed on the device. |
| `osVersion`    | `string`   | :x:                | Operating system version of the device.      |

```C#
/// <summary>
/// DTO to transfer devices's information
/// </summary>
public class DeviceInfoDto
{
    /// <summary>
    /// User's phone number associated with the device. (Required)
    /// </summary> 
    public string Phone { get; set; }

    /// <summary>
    /// Country code associated with the phone number. (Required)
    /// </summary>
    public string CountryCode { get; set; }

    /// <summary>
    /// Unique identifier of the device. (Optional)
    /// </summary>
    public string DeviceID { get; set; }

    /// <summary>
    /// Type of device, e.g., 'Android' or 'iOS'. (Optional)
    /// </summary>
    public string DeviceType { get; set; }

    /// <summary>
    /// Version of the application installed on the device. (Optional)
    /// </summary>
    public string AppVersion { get; set; }

    /// <summary>
    /// Operating system version of the device. (Optional)
    /// </summary>
    public string OsVersion { get; set; }
}

```
