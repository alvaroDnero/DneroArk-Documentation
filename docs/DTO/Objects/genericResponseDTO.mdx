# Generic Response DTO

| Field    | Type   | 
|----------|--------|
| `event`  | String | 
| `message`| String | 


## C#
```c#
/// <summary>
/// Represents a Data Transfer Object (DTO) for an operation result or error message.
/// </summary>
public class GenericResponseDTO
{
    /// <summary>
    /// A code indicating the type of result or error.
    /// </summary>
    public string Event { get; set; }

    /// <summary>
    /// A detailed description providing context for the result of the operation.
    /// </summary>
    public string Message { get; set; }
} 

```
