# User DTO

| Name                  | Type       | Description                            |
|-----------------------|------------|----------------------------------------|
| `userId`              | `long`     | Unique identifier of the user.         |
| `firstName`           | `string`   | User's first name.                     |
| `lastName`            | `string`   | User's last name.                      |
| `imgUrl`              | `string`   | URL of the user's profile image.       |
| `deviceInfo`          | `object`   | Information about the user's device. For detailed information, see the **[Device Info](./deviceInfoDto)**  |

  <TabItem value="c#" label="C#">

```C#
/// <summary>
/// Represents a Data Transfer Object (DTO) for a user.
/// </summary>
public class UserDTO
{
    /// <summary>
    /// Unique identifier of the user.
    /// </summary>
    public long UserId { get; set; }

    /// <summary>
    /// User's first name.
    /// </summary>
    public string FirstName { get; set; }

    /// <summary>
    /// User's last name.
    /// </summary>
    public string LastName { get; set; }

    /// <summary>
    /// URL of the user's profile image.
    /// </summary>
    public string ImgUrl { get; set; }

    /// <summary>
    /// Information about the user's device.
    /// For detailed information, see the DeviceInfoDTO.
    /// </summary>
    public DeviceInfoDTO DeviceInfo { get; set; }
}
```
