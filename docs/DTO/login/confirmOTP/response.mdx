# Generate OTP Response

| Name           | Type     | Description                                    |
|----------------|----------|-----------------------------------------------|
| `userId`       | `Long`   | Id of the user's profile                           |
| `accessToken`  | `string` | Short-lived token containing user information. |
| `refreshToken` | `string` | Long-lived token.                             |
