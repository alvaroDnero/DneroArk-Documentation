Crea el OTP para poder iniciar sesión.

### Header

| Key      | Value     |
|-------------|----------|
| `Content-Type` | application/json |






### JSON Body

| Name      | Type     | Required | Description                          |
|--------------|----------|---------------------|--------------------------------------|
| `phone`      | `string` | :white_check_mark:  | numero de telefono del usuario.
| `countryCode`| `string` | :white_check_mark:  | codigo de pais.     |     |




```JSON
{
  "phone": "7149816149",
  "countryCode": "1"
}

```

### JSON Response: Succes (200)
```JSON
null

```

### JSON Response: Error
```JSON
Número de teléfono no existe en los registros.
```

```JSON
Sesión abierta en otro dispositivo, ciérrala y vuelve a intentarlo.

```


```JSON

```
