Login

POST /DneroArk/login/phone/otp

Confirma el OTP para poder iniciar sesión.

Header​

KeyValue
Content-Typeapplication/json
Acceptapplication/json

JSON Body​

NameTypeRequiredDescription
phonestring✅Numero de telefono del usuario.
countryCodestring✅Código de pais.
otpstring✅Código recibido por OTP.
deviceIDstring❌Id del dispositivo para evitar sesiones abiertas en dispositivos distintos.
{
  "phone": "7149816149",
  "countryCode": "1",
  "otp": "999999",
  "deviceID": "192.168.0.0"
}
