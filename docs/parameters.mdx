# Parameters

**Not all endpoints need all parameters**

| Name           | Type      |  Description                                                                                   |
|----------------|-----------|-----------------------------------------------------------------------------------------------|
| `statuses`     | `int[]`   | An array of statuses to filter the information. If `null` or `0`, all statuses are returned. Multiple statuses can be provided. For detailed information, see the <a href={`./${Paths.coinStatuses}`}>**Status Codes page.**</a> |
| `pageSize`     | `int`     | The number of items to include in each page of the results. If set to `0`, all existing elements will be returned, and `page` will be ignored (treated as `0`). Required if using the <a href={`./${Paths.paginator}`}>**paginator**</a>. |
| `page`         | `int`     | The page number to retrieve, starting from 1 for the first page. Required if using the <a href={`./${Paths.paginator}`}>**paginator**</a>. |
| `sortOrder`    | `enum`    | Specifies the sorting order. Possible values are `ASC` for ascending (oldest to newest) and `DESC` for descending (newest to oldest). Defaults to `DESC`. |
| `verbose`      | `boolean` | If `true`, returns additional details about each object. If `false`, only returns essential information. |
