# User Object Model


| Name                  | Type       | Description                            |
|-----------------------|------------|----------------------------------------|
| `userId`              | `long`   | Unique identifier of the user.         |
| `userName`            | `string`   | First name and last name of the user.  |
| `userImgUrl`          | `string`   | URL of the user's profile image.       |
