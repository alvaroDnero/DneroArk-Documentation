It will return coins with [status 2.](/docs/coinsStatuses),  specifically those coins where the user is the recipient.


The information will be the same as the <a href={`/DneroArk-Documentation/docs/${Paths.coinsAll}`}>**{Endpoint.coinsAll}**</a> endpoint.
