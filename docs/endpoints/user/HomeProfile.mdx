# User


<div className="highlight-box">
  <HttpMethodTag method={method} /> {apiPath}
</div>





Get user profile information.

### Header

| Key      | Value     |
|-------------|----------|
| `Authorization` | Bearer `{{accessToken}}`|



---

### JSON Response: Success

| Name            | Type     | Description                                     |
|-----------------|----------|-------------------------------------------------|
| `userId`        | `long`   | Unique identifier of the user.                  |
| `imgProfileUrl` | `string` | URL where the profile image is hosted.          |
| `firstName`     | `string` | User's first name.                              |
| `lastName`      | `string` | User's last name.                               |
| `phone`         | `string` | User's phone number.                            |
| `dolarBalance`  | `double` | Total dollar balance specific to ARK.           |
| `cryptoBalance` | `double` | Total cryptocurrency balance for ARK.           |
| `pendingCoins`  | `int`    | Number of pending Coins to collect.             |
