# User Object Model

| Name                  | Type       | Description                            |
|-----------------------|------------|----------------------------------------|
| `userId`              | `long`     | Unique identifier of the user.         |
| `firstName`           | `string`   | User's first name.                     |
| `lastName`            | `string`   | User's last name.                      |
| `imgUrl`              | `string`   | URL of the user's profile image.       |
| `deviceInfo`          | `object`   | Information about the user's device. For detailed information, see the **[Device Info](./phoneObject)**  |

<details className='details-custom' open>
 <summary>Example</summary>

```JSON
{

```
