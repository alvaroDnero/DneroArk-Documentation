# Balance Object


| Name            | Type     | Description                                     |
|-----------------|----------|-------------------------------------------------|
| `cashBalance`  | `double` | Total dollar balance specific to ARK.           |
| `cryptoBalance` | `double` | Total cryptocurrency balance for ARK.           |


<details className='details-custom' open>
 <summary>Example</summary>

```JSON
{
  "cashBalance": 1500.75,
  "cryptoBalance": 2.356
}

```
</details>
