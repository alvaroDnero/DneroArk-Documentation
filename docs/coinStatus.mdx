# Coin Statuses


| **Status Code** | **Description**                                                                 |
|:---------------:|---------------------------------------------------------------------------------|
| `0`             | Status zero does not exist, but if used as a <a href={`./${Paths.parameters}`}>**parameter**</a>, it will return coins from all statuses. |
| `1`             | Pending.                                             |
| `2`             | Collected.                                                 |
| `3`             | Expired.                                                              |
