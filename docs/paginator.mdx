# Paginator
## Common properties in the paginator

| Name                  | Type       |   Description                                                                                  |
|-----------------------|------------|-----------------------------------------------------------------------------------------------|
| `currentPage`         | `int`      | The current page number being returned (starting from 1 for the first page).                |
| `totalPages`          | `int`      | The total number of pages available based on the `pageSize` and total records.              |
| `pageSize`            | `int`      | The number of items per page in the paginated results.                                       |
| `totalElements`       | `long`     | The total number of elements available based on the provided filters.                   |




## Specific properties in the paginator: 


### Transactions
| Name                  | Type       |   Description                                                                                  |
|-----------------------|------------|-----------------------------------------------------------------------------------------------|
| `transactions`        | `Array`    | A list containing all the transactions for the requested page.                              |
